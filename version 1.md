# Version 1 Development Plan

The first version focuses on building a minimum viable coding contest platform with core functionality.

---

## Participants

Participants should be able to:

- Register an account
- Login to the platform
- Join a contest
- View contest problems
- Write and submit code
- View their submission history
- View the live leaderboard

---

## Admin

Admins should be able to:

- Create contests
- Add problems to contests
- Add test cases for each problem
- Start or stop contests
- View all submissions
- Manage leaderboard

---

## System Features

The system should provide:

- Contest timer
- Code submission handling
- Code execution through Judge0 API
- Automatic evaluation of submissions
- Score calculation
- Leaderboard ranking

---

## Technology Stack

Backend
- Python
- FastAPI

Database
- SQLite

Frontend
- HTML
- CSS
- JavaScript

Code Execution
- Judge0 API

---

## Version 1 Goal

Build a working contest platform capable of hosting a small programming competition for a college-level event.

The system should support:
- ~100 participants
- multiple problems per contest
- automatic judging
- live leaderboard
